# Starlet Engine

Organization-wide configuration and documentation for the [starlet-engine](https://github.com/starlet-engine) org.

- `profile/README.md` — Public org profile page
- `README.md` — Organization configuration and documentation

## Getting Started

Clone and configure the full mono-repo environment using [`starlet-setup`](https://github.com/starlet-engine/starlet-setup):

```bash
# Example - refer to starlet-setup docs for full usage
starlet-setup starlet-engine/samples
```
